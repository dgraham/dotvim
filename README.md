# Vim configuration

A tiny set of vim customizations.

![MacVim](/images/MacVim.png)

## Install

```
git clone https://github.com/dgraham/dotvim.git ~/.vim
cd ~/.vim
git submodule update --init
ln -s ~/.vim/vimrc ~/.vimrc
```

## Update

```
cd ~/.vim
git pull
git submodule update
```
